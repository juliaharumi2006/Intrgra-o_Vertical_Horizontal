Com base nos conceitos de Sistemas Digitais de Controle Distribuído (SDCD), é possível aplicar essa arquitetura no Projeto Integrador para monitorar e controlar processos industriais de forma eficiente e em tempo real.

Primeiramente, cada componente do sistema possui um papel específico dentro da arquitetura. O sensor de temperatura é responsável por coletar dados do ambiente ou de máquinas, sendo o ponto inicial do sistema. O ESP32 atua como um microcontrolador que recebe esses dados do sensor, realiza um pré-processamento e os envia para a rede. O protocolo MQTT é o responsável pela comunicação, permitindo o envio leve e rápido dos dados, ideal para aplicações de IoT. O servidor recebe, armazena e organiza essas informações, podendo também realizar análises. Já o aplicativo mobile permite que os usuários acessem os dados remotamente, enquanto o dashboard apresenta essas informações de forma visual (gráficos, alertas, indicadores), facilitando a tomada de decisão pela gestão da fábrica.

O fluxo de dados ocorre de forma contínua e integrada. Primeiro, o sensor coleta a temperatura do ambiente ou equipamento. Em seguida, o ESP32 lê esse valor e o envia via MQTT para o servidor. O servidor processa e armazena os dados, disponibilizando-os para o aplicativo e o dashboard. Por fim, os gestores acessam essas informações em tempo real, podendo acompanhar o desempenho dos processos industriais.

Com base nas informações exibidas no dashboard, diversas decisões podem ser tomadas. Por exemplo, é possível realizar o controle de processos, ajustando temperaturas automaticamente para manter a qualidade da produção. Também é possível fazer a prevenção de falhas, identificando variações anormais que indicam problemas em máquinas. Além disso, o sistema permite o monitoramento contínuo de equipamentos, ajudando na manutenção preventiva e evitando paradas inesperadas.

Por fim, o uso de um sistema distribuído traz várias vantagens no contexto industrial. Entre elas estão a maior escalabilidade, já que novos sensores podem ser adicionados facilmente; a flexibilidade, permitindo integrar diferentes tecnologias; a confiabilidade, pois o sistema não depende de um único ponto central; e a resposta em tempo real, essencial para ambientes industriais. Além disso, esse tipo de sistema contribui para a transformação digital das empresas, tornando os processos mais inteligentes e eficientes.

Assim, a aplicação de um SDCD no Projeto Integrador demonstra como a integração entre hardware, software e comunicação pode melhorar significativamente o controle e a gestão industrial.
